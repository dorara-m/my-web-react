## 更新したら

サイトのデザイン大幅変更はメジャーアップデート、項目追加などはマイナーアップデート、見た目の変わらないロジック調整などはパッチアップデートとする。
- release notes に更新した内容を追加する
- footerの日付を更新する
- package.json の version を更新する
